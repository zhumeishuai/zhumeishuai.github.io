    1 9 面向对象

    1.1 面向对象定义

    1.1.1 面向对象介绍

    # 什么是编程：
编程就是程序员用和计算机约定好的语言向计算机描述我们的需求
了解我们要编写的事物的规律，在工作中，我们称之为业务。就是一个好的描述方式，就像我们在小学写作文，有记叙文有议论文，对于程序员来说，所谓的记叙文或议论文就是我们变成的思路，通常由：面向对象，面向过程。

    # 什么是面向对象

面向对象无法就是一种编程思路的表达方式
在变成过程中，把数据及对数据的操作方法放在一起，作为一个相互依存的整体，用这样的思路变成就是面向对象
通过来说就是我们在变成的时候将一类事物的参数和功能描述在一个叫 class 的东西里面，进行编程的思路

    对象：万物皆对象，就是事实存在的事物
类：是对事物的划分，当我们在描述的时候用到类，就是在描述一系列的事物的共性特征
实例：从类中具体映射出来的个体
# 注意：类和实例都有独立的内存空间，相互独立，互不影响，实例来源于类但是要比类更加具有个性
域：属于类或实例的变量，一些名词
方法：属于类或实例的功能 < 函数 >，一般来说是函数，一些动词
属性：域和方法的统称
类方法：属于类的功能
类属性：属于类的属性
实例方法：属于实例的功能
实例属性：属于实例的属性

    # 面向过程的方式：

school= {
    'Linux':[],
    'Python':[]
}
def register(major,student):
    if major in school:
        school[major].append(student)
        print('报名成功')
    else:
        print('抱歉，我们学校没有%s专业'%major)
if __name__ == '__main__':
    student = {
        'name':'hl',
        'age':19,
        'sex':'男',
        'marjor':'Python'
    }
register(student.get('marjor'),student)
print(school)

# 执行结果： 

报名成功
{'Linux': [], 'Python': [{'name': 'hl', 'age': 19, 'sex': '男', 'marjor': 'Python'}]}

    # 面向对象的方式：

class Registration:
    def __init__(self):
        self.school={
            'Linux':[],
            'Python':[]
        }
    def register(self,major,student):
        if major in self.school:
            self.school[major].append(student)
            print('报名成功')
        else:
            print('抱歉，我们学校没有%s专业'%major)
if __name__ == '__main__':
    student = {
        'name':'for',
        'age':19,
        'sex':'男',
        'major':'Python'
    }
reg = Registration()
reg.register(student.get('major'),student)
print(reg.school)

# 执行结果：

报名成功
{'Linux': [], 'Python': [{'name': 'for', 'age': 19, 'sex': '男', 'major': 'Python'}]}

    两种方法得到的结果是一样的，但是考虑到现实生活中，报到是到一个叫报到处的地方办理报到，而不是调用报到的功能。就好比解决吃啤酒鸭的问题
第一种方式 < 面向过程 >：
    1、养鸭子
    2、鸭子长大
    3、杀
    4、佐料
    5、烹饪
    6、吃
第二种方式 < 面向对象 >：
    1、找个卖啤酒鸭的人
    2、给钱
    3、吃
# 所以，面向对象更贴近我们的思维惯性，毕竟在生活中我们周围是一个一个的事物对象

    1.2 类的编写

        Python 变成中类的概念可以比作是某种类型集合的描述，如 '人类' 可以看做一个类，然后用人类这个类定义出每个具体的人-你、我、他等作为其对象
    类还拥有属性和功能，属性即类本身的一些特性，如人类有名字、身高体重等属性，而属性值则会根据每个人的不同而不同；功能则是类所能实现的行为，如人类拥有吃饭、走路睡觉等功能。

    1.2.1 类的定义

    # 类的格式

class 类名 (继承列表):
    '类的文档字符串'

    1.3 实例方法和 self

    1.3.1 实例方法

    # 语法：
    class 类名(继承列表):
        def 实例方法名(self,参数1,参数2,...):
            '文档字符串'
            语句
# 作用： 
    用于描述一个对象的行为，让此类型的全部对象都拥有相同的行为。
# 说明：
    1. 实例方法实质是函数，是定义在类内的函数
    2. 实例方法至少有一个形参，第一个形参代表调用这个方法的实例，一般命名为 'self'
# 实例方法调用语法：
    实例.实例方法名(调用传参)
    类名.实例方法名(实例，调用传参)

    class Cat:
    '我是猫咪'
    def eat(self):
        '猫'
        print('猫在吃......')
    def drink(self):
        print('猫在喝水......')
tom = Cat()     # 创建一个 tom 实例对象
tom.eat()       # 调用实例方法
tom.drink()
Cat.eat(tom)    # 通过类名调用实例方法

# 执行结果：

猫在吃......
猫在喝水......
猫在吃......
