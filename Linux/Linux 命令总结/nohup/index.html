    Nohup

    nohup

    Describe: 在终端退出时使程序忽略 SIGHUP 信号，保证程序能够继续正常进行。SIGHUP 信号是在终端被中止时向它所关联的进程所发出的信号，进程收到这个信号后就会中止运行。默认将输出追加到 ./nohup.out
Usage: nohup COMMAND [ARG]...
  or:  nohup OPTION

    nohup ping baidu.com &> /data/nohup.log &
